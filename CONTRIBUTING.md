# Contributing to Pasty / 페이스티 기여 가이드라인

Thank you for your interest in Pasty! We welcome contributions to improve this tool.
페이스티에 관심을 가져주셔서 감사합니다! 이 도구를 개선하기 위한 기여를 환영합니다.

## How to Contribute / 기여 방법
1. Fork the repository. / 리포지토리를 포크합니다.
2. Create a new branch for your feature or bug fix. / 기능 구현 또는 버그 수정을 위한 새 브랜치를 생성합니다.
3. Ensure all code follows the bilingual documentation and premium aesthetic guidelines. / 모든 코드가 한영병기 문서화 및 프리미엄 미학 가이드라인을 따르는지 확인합니다.
4. Submit a Pull Request. / 풀 리퀘스트를 제출합니다.

## Code of Conduct / 행동 강령
Please be respectful and collaborative. / 서로를 존중하고 협력적인 태도를 유지해 주세요.
