# Contributing to Pasty / Pasty 기여하기

## How to Contribute / 기여 방법

### Reporting Issues / 이슈 보고
- Use GitHub Issues to report bugs or request features
- Provide detailed steps to reproduce bugs
- Include system information (OS, Python version)

### Pull Requests / 풀 리퀘스트
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a PR with clear description

### Development Setup / 개발 환경 설정
```bash
# Clone
git clone https://github.com/hslcrb/pasty.git
cd pasty

# Virtual environment
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate  # Windows

# Dependencies
pip install -r requirements.txt
```

### Code Style / 코드 스타일
- Follow PEP 8
- Use meaningful variable names
- Add comments for complex logic
- Keep functions focused and small

### Testing / 테스트
- Test on multiple platforms (Linux, Windows, macOS)
- Test both light and dark themes
- Test both Korean and English languages
- Verify settings persistence

### Documentation / 문서화
- Update README.md for new features
- Update TIMELINE.md for significant changes
- Keep inline comments up to date
